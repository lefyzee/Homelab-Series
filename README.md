# Security Daemons Homelab Series (DePaul)

This repository contains all documentation, slides, configs, and other materials for the Security Daemons 10-week Homelab Series at DePaul University.

The goal of this series is to help students build practical, hands-on skill in homelabbing using tools that can be replicated on personal hardware or DePaul SecLabs infrastructure.

---


## Series Overview

**Weeks and Topic***

1. Homelab Inspiration and VMware Basics
2. Building Your First Debian Server
3. Hosting Your Own Minecraft Server
4. Internal Website Hosting with Apache
5. Tailscale and Pi-Hole Setup
6. Catch-up & Open Lab Week
7. Setting up a NAS (TrueNAS in VMware)
8. SIEMS and Logging
9. Monitoring and Dashboards
10. Homelab Capstone Showcase


Each week includes:

- A lab guide ('docs/weekXX/guide.md')
- Optional instructor notes for the presenter
- Slides ('slides/weekXX/')
- Config files or scripts when applicable

---

## Requirements

To follow along with most labs, you will need **one** of:

- A computer capable of running **VMware Workstation** or
- Access to DePaul's **SecLabs remote VMs** (instructions provided separately)

Some labs may also use:

- Debian, PI OS, Ubuntu Server IOSs
- TrueNAS ISO
- Basic familiarity with the command line
